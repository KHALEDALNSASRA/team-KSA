Nature sounds

The <audio> element is used to add sound to a webpage. It allows users to play, pause, and control audio directly in the browser.

Nature video

The <video> element is used to display videos on a webpage. It provides built-in controls for playing, pausing, adjusting the volume, and viewing the video in fullscreen.

Embedded Content

The <iframe> element is used to embed external content, such as YouTube videos, Google Maps, or other webpages, inside the current webpage.
